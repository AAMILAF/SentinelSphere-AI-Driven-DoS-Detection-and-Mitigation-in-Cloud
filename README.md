# SentinelSphere-AI-Driven-DoS-Detection-and-Mitigation-in-Cloud

SentinelSphere is an intelligent cloud security framework that detects and mitigates Denial-of-Service (DoS) attacks using Large Language Models (LLMs), NGINX traffic analytics, Prometheus monitoring, and automated mitigation policies.

The framework integrates:

- ApacheBench attack simulation
- NGINX traffic monitoring
- Ollama-based local LLM inference
- Mistral-7B AI model
- Automated mitigation engine
- Prometheus metrics collection
- Grafana visualization
- Real-time web dashboards

The system continuously analyzes cloud traffic patterns and automatically activates protection mechanisms when malicious behavior is detected.
